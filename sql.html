7.ALTER Command

ADD COLUMN

8.GRANT Command

syntax: GRANT privileges_names ON object TO user;
Example:
GRANT SELECT, INSERT, DELETE, UPDATE ON Users TO 'Amit'@'localhost;

9.REVOKE Command

syntax: REVOKE privileges ON object FROM user;
Example:
REVOKE SELECT, INSERT, DELETE, UPDATE ON Users FROM 'Amit'@'localhost;

10.COMMIT Command

syntax: COMMIT;
Example:
DELETE FROM persons where person_id=15; commit;

11.ROLLBACK Command

syntax: Rollback;
Example:
DELETE FROM persons where person_id=15;
Rollback;
